# My Fit Journal（React + TypeScript 后端）

这是一个基础的个人健身记录网站：

- 前端：React
- 后端：Node.js + TypeScript（源码在 `server/src/index.ts`）
- 功能：PR 记录、补剂/工具评价

## 启动

```bash
npm start
```

打开：

- 前端页面：<http://localhost:4000>
- API：`/api/prs`、`/api/reviews`

## TypeScript 构建（可选）

```bash
npm run build:server
```

> 当前仓库已包含可直接运行的编译产物：`server/dist/index.js`。
